    DMDirc-{{ site.versions.stable }}.exe

    DMDirc-{{ site.versions.stable }}.deb

    DMDirc-{{ site.versions.stable }}.rpm

    For Red Hat, Fedora, etc

  • Download the OS X package

    DMDirc-{{ site.versions.stable }}.dmg

    For Mac OS X

  • Download the Java archive

    DMDirc-{{ site.versions.stable }}.jar

    For advanced users, on any platform that supports Java

Apt repository

Users of Debian-based distros can also use our apt repository to stay up to date automatically:

    sudo wget http://www.dmdirc.com/apt/dmdirc.list -O /etc/apt/sources.list.d/dmdirc.list
    wget -qO- http://www.dmdirc.com/apt/dmdirc.gpg.key | sudo apt-key add -
    sudo apt-get update
    sudo apt-get install dmdirc
