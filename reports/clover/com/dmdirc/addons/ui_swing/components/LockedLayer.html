41    /**
42    * An implementation of the {@code BufferedLayerUI} which provides a lightweight disabling for the
43    * content of its {@link JXLayer}. This allows temporarily blocking a part of the interface with all
44    * it subcomponents, it is also useful when some kind of action is in progress, e.g. reading data
45    * from a database.
46    * <p>
47    * When {@code true} is passed to the {@link #setLocked(boolean)}, the {@code JXLayer} of this
48    * {@code LockableLayerUI} becomes "locked". It sets the "wait" mouse cursor and stops reacting on
49    * mouse, keyboard and focus events.
50    * <p>
51    * If {@code setLocked(boolean)} is called with {@code false} parameter after that, the
52    * {@code JXLayer}, together with all its subcomponents, gets back to live.
53    * <p>
54    * Subclasses usually override {@link #paintLayer(Graphics2D,JXLayer)} or
55    * {@link #getLayerEffects(JXLayer)} to implement some visual effects when {@code JXLayer} is in
56    * locked state.
57    * <p>
58    * Here is an example of using {@code LockableLayerUI}:
59    * <pre>
60    * JComponent myComponent = getMyComponent(); // just any component
61    *
62    * LockableLayerUI lockableUI = new LockableLayerUI();
63    * JXLayer&lt;JComponent&gt; layer = new JXLayer&lt;JComponent&gt;(myComponent,
64    * lockableUI);
65    *
66    * // locking the layer, use lockableUI.setLocked(false) to unlock
67    * lockableUI.setLocked(true);
68    *
69    * // add the layer to a frame or a panel, like any other component
70    * frame.add(layer);
71    * </pre>
72    *
73    * @param <T> Generic type
74    */
